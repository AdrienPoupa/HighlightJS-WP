Lightest possible integration for HighlightJS in WP.

== Description ==

Loads Highlight JS in WordPress in the simplest manner.

Ships the latest version of HighlightJS (see https://highlightjs.org/).

To create a code snippet in Gutenberg, create a new paragraph and type three backticks: ```

Simply paste your code within the new code block and HighlightJS will highlight your code automatically.

== Installation ==

Upload the HighlightJS WP plugin to your blog and activate it.
